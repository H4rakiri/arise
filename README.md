# 「ARISE」

Персональный трекер задач с геймификацией жизни в эстетике Системы (Solo Leveling).
Реализация по ТЗ v1.0.

## Запуск локально

```bash
npm install
npm run dev
```

## Деплой на GitHub Pages

1. Создай публичный репозиторий, запушь этот код в `main`.
2. В настройках репо: **Settings → Pages → Source: GitHub Actions**.
3. Workflow `.github/workflows/deploy.yml` соберёт и выложит сайт автоматически.

## Синхронизация данных (Mac ↔ десктоп)

1. Создай **приватный** репозиторий под данные, например `user/arise-data`.
2. Сгенерируй fine-grained personal access token: **GitHub → Settings → Developer settings →
   Fine-grained tokens** — доступ только к этому репо, permission **Contents: Read and write**.
3. На сайте: **「SYSTEM」 → 「SYNC」** — введи `user/arise-data` и токен, нажми «Подключить».

Дальше всё само: каждое изменение (закрыл квест, выпала карта) через ~3 секунды коммитится
в `data.json`. Токен хранится только в `localStorage` браузера, в репозиторий не попадает.
При оффлайне данные живут в локальном кеше и уедут при следующем подключении.

## Нейронка (Фаза 5)

**「SYSTEM」 → 「NEURAL」** — включи тумблер. Нужен WebGPU (Vivaldi/Chromium — ок).
При первом разборе «плана дня» модель Qwen2.5-1.5B (~1 ГБ) скачается и закешируется в браузере.
Без WebGPU или с выключенным тумблером разбор плана работает на встроенной эвристике.

## Тюнинг формул

Все константы (XP, кривая уровней, ранги, редкости, заморозки) — в [src/config.js](src/config.js).
